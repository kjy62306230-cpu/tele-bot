Daily flow

사촌의 하루는 이렇게 흘러갑니다

시스템은 사촌이 자고 있을 때부터 일하기 시작합니다. 사촌이 해야 할 일은 폰을 보는 것뿐입니다.

06:00KST
시스템이 자동으로 가동
사촌은 아직 자고 있음. n8n 워크플로가 알아서 미국장 데이터를 수집하고 한국 관련주를 매칭한 뒤, AI가 한국어 리포트를 생성합니다.
07:00KST
텔레그램에 알림 도착
출근 준비하면서 폰으로 확인. 표·뉴스·한국 관련주가 정리된 리포트를 5분 안에 훑어봅니다.
08:00KST
출근길에 정독
버스·지하철에서 자세히 읽으며 오늘 주목할 종목을 머릿속으로 정리합니다.
09:00KST
한국장 개장 — 직접 매매
증권사 앱으로 본인이 직접 매수·매도. 시스템은 정보만 제공하고, 결정은 100% 사촌의 판단입니다.
09:30~
회사 일과 시작
컴퓨터를 못 만져도 괜찮습니다. 이미 아침에 모든 정보를 받았기 때문에 따로 챙길 게 없습니다.
06:00+ 1d
자동으로 다시 반복
사촌이 손댈 일 없습니다. 매일 새 리포트가 텔레그램에 알아서 들어옵니다.

Roles

누가 뭘 하는가

사촌은 거의 아무것도 하지 않습니다. 시스템 운영은 J가 맡습니다.

Cousin · receiver
받기 · 보기 · 매매
5–10 min / day
  • 아침 7시 텔레그램 확인 (출근 준비하면서)
  • 9시 한국장 개장 시 본인 판단으로 매매
  • 이상하면 J에게 카톡 (예: "어제 매칭 잘못된 거 같아")
J · operator
시스템 운영 · 개선
≤ 1 hr / month
  • 매일 텔레그램 발송 정상 확인 (1분)
  • 주 1회 매칭 실패 종목 보강 (10분)
  • 월 1회 API 사용량·비용 점검 (5분)
  • 분기 1회 사촌 피드백 반영해 프롬프트 개선 (30분)
04
Scope

시스템이 하는 것 · 안 하는 것

시작 전에 명확히 합의하고 가야 합니다. 친척 사이일수록 더욱.

+
Does
  • 미국장 데이터 자동 수집나스닥 TOP 10 + 거래량 + 상승 원인
  • 한국 관련주 매칭 정보 제공공급망 1·2단계 + 동일 섹터 + 자회사
  • 주요 뉴스 영향도 분석한국 시장 영향도 순으로 5–7건
  • 매일 정해진 시간 자동 발송매일 07:00 KST · Telegram
−
Does not
  • 매수·매도 추천"이거 사세요" 같은 추천 절대 없음
  • 주식 자동 매매주식은 알림만, 매매는 본인이 직접
  • 수익 보장판단·결정·손익은 100% 사촌 책임
  • 100% 정확한 매칭목표 95%+, "확인 필요" 표기 활용
▲ Pre-agreement
이 시스템은 정보 제공 도구일 뿐입니다. 모든 매매 판단·결정·손익은 사촌 본인의 책임이며, 시스템 운영자(J)는 매칭 정확도 향상과 시스템 안정 운영에만 책임을 집니다.
05
Why it works

사촌에게 좋은 3가지 이유

왜 이 시스템이 사촌의 라이프스타일에 맞는가.

01
새벽에 안 일어나도 됨
자는 동안 시스템이 데이터 수집과 분석을 마쳐둡니다. 일어나면 결과만 보면 끝.
02
회사 컴퓨터 안 만져도 OK
폰만 있으면 됩니다. 점심시간이나 출근길에 5분만 투자해도 그날 매매 판단이 가능합니다.
03
본인 판단력은 그대로
AI는 정보를 정리할 뿐, 결정은 사촌이 합니다. 본인의 트레이딩 감각이 핵심.
